# Pharmacy Inventory & Billing System

### 🚀 Description
A standalone Java application designed to automate pharmacy operations. It tracks medicine stock, manages expiry dates, and generates precise customer bills with GST and discounts.

### ✨ Key Features
- **Fast Search:** Uses HashMap for O(1) time complexity in book retrieval.
- **Automated Billing:** Calculates 10% GST and 5% Discount automatically.
- **Low Stock Alert:** Triggers alerts when medicine stock falls below 5.
- **Custom Exceptions:** Handles errors like `MedicineNotFound` and `OutOfStock`.

### 🛠️ Tech Stack
- **Language:** Java (JDK 8 or above)
- **Concepts:** OOPs (Encapsulation, Inheritance), Java Collections Framework, Exception Handling.

### 🏁 How to Run
1. Clone this repository.
2. Compile: `javac PharmacyManagement.java`
3. Run: `java PharmacyManagement`
